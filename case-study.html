Enugu Wi-Fi Project

The Enugu SME centre project provides affordable wi-Fi zones across the state as part of the youth empowerment scheme sponsored by the Enugu state Government.

Through this project, youths are empowered with tools to help them generate additional sources of income by reselling internet data at affordable rates. This project is white-labelled and managed by Wicrypt technology and has 118 hosts/agents and more than 5,000 users.
